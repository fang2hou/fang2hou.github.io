自定义 SSH 的服务端口

发表于 2017-02-03 更新于 2018-11-13 分类于 通信网络

契机

每次在维护服务器的时候，总会在登录时收到许多其他 IP 尝试登陆 SSH 的失败信息。为了提高服务器的安全性，修改 SSH 默认的 22 端口可能会帮助屏蔽一些恶意扫描。

修改方法
